<h3>安卓微信浏览器Bug总结：</h3>
<ol>
	<li>响应式不支持min-device-width,要用min-width。</li>
	<li>vw单位支持有bug，元素宽度不支持vw。</li>
	<li>css3 animation动画keyframes开始点必须写0%，不能写0。</li>
</ol>
<h3>其他浏览器Bug总结：</h3>
<ol>
	<li>在安卓 4.3 及以下版本是不支持 vw, vh 单位的，UC所有版本均不支持（20151122）。</li>
	<li>UC 对于margin-top 百分比计算有误。</li>

</ol>
